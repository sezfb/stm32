SOURCE_DIRS¶

Added in version 4.4.

Semicolon-separated list of base directories of the target's default source set (i.e. the file set with name and type SOURCES). The property supports generator expressions.

This property is normally only set by target_sources(FILE_SET) rather than being manipulated directly.

See SOURCE_DIRS_<NAME> for the list of base directories in other source sets.
