CTEST_PRESETS_FILE¶

Added in version 4.4.

Specify the PRESETS_FILE option for all relevant ctest(1) Dashboard Client step commands in a dashboard script, or on the ctest command line via the -D option.

This variable sets the PRESETS_FILE option for ctest_configure(), ctest_build(), ctest_test(), and ctest_memcheck() when those commands are not given an explicit PRESETS_FILE argument.

See also CTEST_PRESET, CTEST_CONFIGURE_PRESET, CTEST_BUILD_PRESET, and CTEST_TEST_PRESET.
