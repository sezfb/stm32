JOB_POOL_COMPILE¶

Added in version 4.4.

Ninja only: Pool used for compiling.

The number of parallel compile processes could be limited by defining pools with the global JOB_POOLS property and then specifying here the pool name.

This allows to override the JOB_POOL_COMPILE and JOB_POOL_COMPILE values for specific source files within a same target.

For instance:

set_property(FILE_SET my_fileset TYPE SOURCE PROPERTY JOB_POOL_COMPILE two_jobs)

This property is undefined by default.
